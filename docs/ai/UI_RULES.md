# UI_RULES.md

---

# Design Style

统一风格：

- Premium Mobile UI
- Apple 风格间距
- 极简
- 现代化
- 商业 App 风格

---

# Mandatory Rules

必须遵守：

- 所有颜色使用 AppColors
- 所有间距使用 AppSpacing
- 所有主题使用 AppTheme
- 不允许写死颜色
- 不允许写死 padding
- 不允许页面使用魔法值

---

# Layout Rules

推荐：

- 页面 padding = 16
- 卡片圆角 = 16
- 统一间距系统
- 层级清晰
- 少量阴影
- 避免视觉拥挤

---

# Widget Rules

必须：

- Widget 尽量 const
- StatelessWidget 优先
- 单一职责
- 避免嵌套超过 4 层
